Resolución: Problema de Concesiones Argentina 2000 SRL

## Requisitos de Software
* **Versión mínima de Python:** Python 3.8 o superior.
* **Bibliotecas requeridas:**
  - `pulp`: Para el modelado y resolución del sistema de Programación Lineal Entera.

## Instrucciones de Instalación y Ejecución
1. Abrir la consola.
2. Instalar la dependencia matemática: `pip install pulp`
3. Colocar el script `programacion_lineal.py` en el directorio de trabajo.
4. Ejecutar el script: `python3 programacion_lineal.py`
5. Salida generada: El programa creará automáticamente el archivo `pl_resultado.txt` en la misma carpeta, el cual contiene el reporte con los resultados de la solución óptima
